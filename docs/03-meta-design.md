# 메타 게임 디자인

## 성장 시스템

### 아들 성장 (간접 육성)

부모는 아들을 직접 키울 수 없다. 아들의 성장은 **부모가 조성한 환경**에 의해 간접적으로 이루어진다.

#### 성장 경로

| 성장 수단 | 부모의 기여 | 아들의 행동 | 성장 효과 |
|----------|-----------|-----------|----------|
| 허수아비 훈련 | 허수아비 존재 (상시) | 자율적으로 훈련 | EXP +8~12 / 회당 |
| 모험 전투 | 장비/음식/포션 준비 | 자동 전투 | EXP (전투 결과에 비례) |
| 독서 | 책을 책상에 배치 | 30% 확률로 독서 선택 | 랜덤 스탯 +1 |
| 식사 | 고급 음식 제작 → 식탁 배치 | 배고프면 자동 식사 | 임시 스탯 버프 (고급 음식) |
| 포션 | 영약 제조 → 선반 배치 | 출발 직전 자동 음용 | 임시 스탯 버프 (1모험) |

#### 성장 마일스톤

아들의 레벨이 특정 구간에 도달하면 새로운 시스템이 해금되어 부모의 활동 폭이 넓어진다.

| 아들 레벨 | 해금 요소 | 설명 | 영향 |
|----------|----------|------|------|
| 1 (시작) | 대장간 기본 제작, 주방 기본 요리, 농장 | 나무 검/가죽 갑옷/빵/야채 수프 제작 가능 | 최소한의 준비로 초급 모험 가능 |
| 3 | 연금술 작업대 해금 | 체력 포션, 기본 영약 제조 가능 | 포션으로 생존력 보강 |
| 5 | 장비 강화 해금 | 기존 장비에 강화석 투입하여 스탯 업 | 장비 교체 없이 점진적 강화 가능 |
| 5 | 철 장비 레시피 해금 | 철 검, 철 갑옷 제작 가능 | 중급 장비 라인 접근 |
| 8 | 장비 가챠 해금 | 특수 광석으로 랜덤 등급 장비 획득 | 고등급 장비 획득 루트 추가 |
| 8 | 고급 요리/영약 레시피 해금 | 고기 스튜, 영웅 도시락, 만능 비약 등 | 강력한 버프 제공 가능 |
| 10 | 농장 확장 (4칸 → 6칸) | 밭 2칸 추가 | 재료 자급 효율 대폭 상승 |
| 12 | 미스릴 장비 레시피 해금 | 미스릴 검, 미스릴 갑옷 제작 가능 | 상위 장비 라인 접근 |
| 15 | 포션 선반 확장 (3칸 → 5칸) | 포션 슬롯 2개 추가 | 더 많은 버프 중첩 가능 |

### 장비 성장 체계

장비는 **등급(Rarity)**과 **강화 단계(Enhancement)**라는 두 축으로 성장한다.

#### 등급 시스템

| 등급 | 색상 | 기본 스탯 배율 | 획득 경로 |
|------|------|--------------|----------|
| 일반 (Common) | ⬜ 회색 | x1.0 | 제작, 가챠(60%) |
| 고급 (Uncommon) | 🟩 초록 | x1.3 | 가챠(25%) |
| 희귀 (Rare) | 🟦 파랑 | x1.7 | 가챠(12%) |
| 영웅 (Epic) | 🟪 보라 | x2.2 | 가챠(3%) |

- **제작 장비**: 항상 일반 등급으로 생성된다.
- **가챠 장비**: 랜덤 등급으로 생성되며, 고급 이상 등급은 가챠를 통해서만 획득 가능하다.
- **강화**: 등급과 무관하게 모든 장비를 +5까지 강화할 수 있다.

#### 장비 성장 공식

최종 장비 스탯 = `기본 스탯 x 등급 배율 x (1 + 강화 보너스)`

**예시: 철 검 (STR +7 기본)**

| 등급 | 강화 | STR 최종값 | 전투력 기여 |
|------|------|----------|-----------|
| 일반 | +0 | 7 | 14.0 |
| 일반 | +3 | 9.45 | 18.9 |
| 고급 | +0 | 9.1 | 18.2 |
| 고급 | +3 | 12.3 | 24.6 |
| 희귀 | +0 | 11.9 | 23.8 |
| 영웅 | +0 | 15.4 | 30.8 |
| 영웅 | +5 | 27.7 | 55.4 |

### 레시피 해금 체계

레시피는 **아들 레벨 달성** 또는 **모험 보상 드롭**으로 해금된다.

| 레시피 카테고리 | 초기 보유 | 레벨 3 | 레벨 5 | 레벨 8 | 레벨 12 | 드롭 획득 |
|--------------|---------|--------|--------|--------|---------|---------|
| 대장간 | 나무 검, 가죽 갑옷, 부적 | - | 철 검, 철 갑옷, 민첩의 반지 | - | 미스릴 검, 미스릴 갑옷, 생명의 펜던트 | - |
| 주방 | 빵, 야채 수프 | - | 과일 파이 | 고기 스튜, 영웅의 도시락 | - | 특수 레시피 (보스 드롭) |
| 연금술 | - | 체력 포션, 힘의 영약 | 수호의 영약, 신속의 영약 | 지혜의 영약, 만능 비약 | - | 특수 레시피 (보스 드롭) |

## 보상 구조

### 모험 보상 총괄

모험에서 돌아온 아들은 전투 결과에 따라 보상을 가져온다. 보상은 다음 크래프팅 사이클의 재료로 순환된다.

#### 기본 보상 (전투 1회당)

| 보상 아이템 | 드롭 확률 | 수량 | 사용처 |
|------------|----------|------|--------|
| 골드 | 100% | 5~15 x 레벨 보정 | 범용 재화 (제작, 강화, 씨앗 구매) |
| 몬스터 이빨 | 40% | 1~2 | 연금술 재료 (영약 류) |
| 몬스터 껍질 | 35% | 1~2 | 연금술 재료 (수호의 영약) |
| 가죽 | 30% | 1~2 | 대장간 재료 (방어구 계열) |
| 철광석 | 25% | 1~2 | 대장간 재료 (철 장비) |
| 약초 (랜덤 색) | 45% | 1~3 | 연금술 / 농장 대체 재료 |
| 씨앗 (랜덤) | 20% | 1 | 농장 경작 |
| 고기 | 35% | 1~2 | 주방 재료 (고기 스튜 등) |

#### 희귀 보상 (보스/특수 조건)

| 보상 아이템 | 획득 조건 | 수량 | 사용처 |
|------------|----------|------|--------|
| 미스릴 | 보스 처치 (50%) | 1 | 상위 장비 제작 (미스릴 계열) |
| 보석 | 보스 처치 (40%) | 1 | 악세서리 제작 |
| 강화석 | Lv5+ 전투 승리 (15%) | 1 | 장비 강화 핵심 재료 |
| 특수 광석 | Lv8+ 보스 처치 (30%) | 1 | 장비 가챠 재료 (3개 = 1회) |
| 레시피 | Lv5+ 보스 처치 (20%) | 1 | 새로운 제작법 해금 |
| 책 | 보스 처치 (25%) | 1 | 아들 독서 → 랜덤 스탯 +1 |

#### 레벨 구간별 보상 기대치 (모험 1회 = 전체 전투 합산)

| 레벨 구간 | 전투 횟수 | 골드 기대치 | 일반 소재 기대치 | 희귀 소재 기대치 | 보스 보상 확률 |
|----------|----------|-----------|---------------|---------------|-------------|
| 1~3 | 2~3회 | 30~90 | 3~8종 | 0~1종 | 10% |
| 4~7 | 3~4회 | 80~210 | 5~12종 | 0~2종 | 15% |
| 8~12 | 4~5회 | 180~450 | 8~16종 | 1~3종 | 20% |
| 13~17 | 5~6회 | 350~810 | 10~20종 | 2~4종 | 25% |
| 18~20 | 6~7회 | 600~1260 | 12~24종 | 3~5종 | 30% |

#### 모험 실패 시 보상

아들이 모험 중 HP가 0이 되어 실패 귀환하더라도 **그때까지 획득한 보상의 50%**를 가져온다.

| 항목 | 성공 귀환 | 실패 귀환 |
|------|----------|----------|
| 골드 | 100% | 50% |
| 소재 | 100% | 50% (반올림) |
| EXP | 100% | 전투한 횟수분만 (감소 없음) |
| 희귀 소재 | 100% | 0% (보스 미도달 시) |

**설계 의도**: 실패해도 완전한 손실이 아니므로 좌절감을 줄인다. 하지만 희귀 소재는 성공해야만 얻을 수 있으므로 성공 동기를 유지한다.

## 진행 흐름

### 전체 진행 구조

게임의 진행은 **"모험 사이클"** 단위로 측정된다. 1사이클 = 준비 → 아들 집 생활 → 출발 → 모험 → 귀환.

```
[초기 상태]
  나무 검 + 낡은 가죽옷 + 빵 2개
  아들 Lv.1 / 전투력 ~20
       │
       ▼
[사이클 1~2: 생존 기반 구축] ──────────────────
  목표: 첫 모험 성공, 기본 경제 루프 확립
  부모: 밀 농사 → 빵 생산, 나무 검 제작
  아들: Lv.1→3, 허수아비 훈련 + 초급 모험
  해금: 연금술 작업대 (Lv.3)
       │
       ▼
[사이클 3~4: 장비 전환기] ─────────────────────
  목표: 철 장비로 전환, 포션 활용 시작
  부모: 철 검/갑옷 제작, 체력 포션 양조, 다양한 작물 경작
  아들: Lv.3→5, 중급 모험 도전
  해금: 장비 강화 (Lv.5), 철 장비 레시피 (Lv.5)
       │
       ▼
[사이클 5~7: 강화 심화기] ─────────────────────
  목표: 장비 강화, 고급 음식/포션 활용
  부모: 장비 +2~3 강화, 고기 스튜/영웅 도시락, 다양한 영약
  아들: Lv.5→8, 보스급 몬스터 도전
  해금: 가챠 (Lv.8), 고급 레시피 (Lv.8)
       │
       ▼
[사이클 8~10: 도약기] ────────────────────────
  목표: 미스릴 장비 확보, 농장 최적화
  부모: 미스릴 장비 제작, 가챠 도전, 농장 확장 활용
  아들: Lv.8→12, 고급 모험
  해금: 농장 확장 (Lv.10), 미스릴 레시피 (Lv.12)
       │
       ▼
[사이클 11~15: 최적화기] ─────────────────────
  목표: 최적 빌드 구축, 고강화 도전
  부모: +4~5 강화 도전, 영웅급 가챠, 만능 비약 양산
  아들: Lv.12→18, 최상위 모험
  해금: 포션 선반 확장 (Lv.15)
       │
       ▼
[사이클 16~18: 엔드게임] ─────────────────────
  목표: 아들을 진정한 영웅으로!
  부모: 영웅 등급 장비 +5 강화, 최고급 버프 총동원
  아들: Lv.18→20, 최종 보스 도전
  달성: "진정한 영웅" 마일스톤 → 엔딩
```

### 프로토타입 진행 목표 (3~5사이클)

프로토타입에서는 **사이클 1~5**까지의 경험을 제공한다. 목표 플레이 시간: **약 20~30분**.

| 사이클 | 소요 시간(예상) | 핵심 체험 | 부모 행동 요약 |
|--------|--------------|----------|-------------|
| 1 | 5~6분 | 첫 모험, 아들의 첫 편지 | 빵 굽기 → 식탁 배치 → 아들 출발 관찰 → 편지 읽기 → 귀환 |
| 2 | 5~6분 | 보상 재투자, 농장 활용 | 보상 재료로 장비 제작 → 농사 시작 → 더 나은 음식 제공 |
| 3 | 5~7분 | 연금술 해금, 포션 첫 사용 | 연금술 작업대 접근 → 체력 포션 제조 → 포션 선반 배치 |
| 4 | 5~7분 | 철 장비 전환, 강화 첫 도전 | 철 검/갑옷 제작 → +1 강화 → 고급 음식 제공 |
| 5 | 6~8분 | 보스 조우, 마일스톤 달성 | 최적 장비 + 포션 + 음식 풀 세팅 → 아들 Lv.5 달성 |

## 재화 시스템

### 재화 종류 및 역할

| 재화 | 아이콘 | 주 획득처 | 주 소비처 | 역할 |
|------|--------|---------|---------|------|
| 골드 | 🪙 | 모험 보상 | 제작, 강화, 씨앗 구매 | 범용 화폐. 모든 제작에 소량 필요 |
| 몬스터 이빨 | 🦷 | 모험 드롭 | 연금술 (영약 재료) | 버프 포션 핵심 재료 |

| 사과 | 🍎 | 농장 수확 | 주방 (파이, 도시락) | 고급 요리 재료 |

### 경제 핵심 병목 (의도적 희소성)

게임 경제의 재미는 **"무엇을 먼저 만들 것인가"** 선택에서 나온다. 아래 재화는 의도적으로 희소하게 설계한다.

| 병목 재화 | 병목 구간 | 설계 의도 |
|----------|---------|----------|
| 철광석 | Lv.3~7 | 철 장비로 전환할 때 부족 → 무기 vs 방어구 선택 강요 |
| 강화석 | Lv.5~12 | 강화 시작 후 항상 부족 → 어떤 장비를 강화할지 선택 |
| 미스릴 | Lv.8~15 | 보스에서만 드롭 → 보스 도전 동기 부여 |
| 특수 광석 | Lv.8+ | 가챠 1회에 3개 필요 → 모험 3~4회 축적 후 가챠 |
| 고기 | Lv.5~8 | 고급 음식 핵심 재료지만 농장 불가 → 모험 의존 |
| 씨앗 (비기본) | 전 구간 | 당근/사과/푸른약초 씨앗 등 모험 드롭 의존 |

## 리텐션 요소

### 1. 편지 시스템 (감정적 연결)

아들이 모험 중 보내오는 편지는 단순 정보 전달을 넘어 **부모로서의 감정적 몰입**을 유도한다.

| 편지 유형 | 감정 효과 | 리텐션 역할 |
|----------|----------|-----------|
| 전투 결과 편지 | 뿌듯함 / 걱정 | "내가 만들어준 검으로 이겼다!" → 제작 보람 |
| HP 낮음 편지 | 긴장, 안타까움 | "다음엔 더 좋은 걸 만들어줘야지" → 준비 동기 |
| 보물 발견 편지 | 기대감 | 귀환 보상에 대한 호기심 유지 |
| 보스 조우 편지 | 극도의 긴장 | "우리 아들이 괜찮을까..." → 감정적 투자 |
| 감사 편지 | 감동 | "엄마/아빠가 만들어준 밥이 제일 맛있어요" → 유대감 |

### 2. 장비 수집 / 강화 (수집 욕구)

| 요소 | 리텐션 메커니즘 |
|------|-------------|
| 등급 수집 | 같은 장비라도 고급/희귀/영웅 등급을 모으고 싶은 욕구 |
| 강화 도전 | +3 성공 → +4 도전 → 실패 → 재도전 의지 (강화석 모으기) |
| 슬롯 최적화 | 무기+방어구+악세서리 3슬롯 최적 조합 탐색 |

### 3. 레시피 발견 (탐색 재미)

- 레벨 해금 레시피: 명확한 단기 목표 제공 ("Lv.5 되면 철 검 만들 수 있어!")
- 보스 드롭 레시피: 희소성으로 인한 발견의 기쁨 ("드디어 비밀 레시피가!")
- 레시피 도감 UI에서 미해금 항목이 "???" 로 표시 → 해금 욕구 자극

### 4. 농장 확장 (소유 욕구)

| 확장 단계 | 밭 칸수 | 해금 조건 | 감정 효과 |
|----------|--------|---------|----------|
| 초기 | 4칸 | 시작 | "겨우 돌아가는 텃밭" |
| 중기 | 6칸 | Lv.10 | "드디어 넓어졌다!" 효율 체감 |
| (미래 확장) | 8칸+ | Lv.15+ | 프로토타입 스코프 외 |

### 5. 아들 마일스톤 (성취감)

| 마일스톤 | 레벨 | 연출 | 감정 효과 |
|---------|------|------|----------|
| 첫 보스 처치 | 5 | 특별 편지 + 트로피 아이템 | "우리 아들이 해냈다!" 뿌듯함 |
| 모험가 인정 | 10 | 길드 등록증 편지 | "어느새 이렇게 컸구나" 감동 |
| 소문난 영웅 | 15 | 마을 사람 감사 편지 | 부모로서의 보람 |
| 진정한 영웅 | 20 | 왕의 훈장 + 엔딩 연출 | 완결의 만족감 |

### 6. 시간 압박 / 긴장감 루프 (세션 유지)

- 배고픔 시스템: 아들의 배고픔이 분당 -5 → 만복(100)에서 출발 조건(80)까지 약 4분 → 부모는 **4분 안에 가능한 한 많이 준비**해야 함
- 이 시간 압박이 "한 번 더" 플레이를 유도: "이번엔 포션도 만들어놔야지" → 다음 사이클로 자연스럽게 이어짐

## 코어 시스템 접점

### 코어 → 메타 (모험 결과가 경제로 유입)

| 코어 이벤트 | 메타로 전달되는 데이터 | 메타 시스템 반응 |
|------------|---------------------|---------------|
| 전투 승리 (완승/승리/신승) | EXP, 골드, 드롭 아이템 | 아들 경험치 축적, 재화 인벤토리 추가 |
| 전투 패배 | 감소된 EXP, 감소된 보상 | 최소한의 재화 확보 (완전 손실 방지) |
| 보스 처치 | 희귀 재료, 레시피, 책 | 새 제작 경로 해금, 독서 콘텐츠 추가 |
| 아들 레벨업 | 레벨 수치 | 마일스톤 체크 → 시스템/레시피 해금 |
| 모험 완료 | 보상 총 목록 | 인벤토리 일괄 추가, 다음 사이클 재료 확보 |
| 허수아비 훈련 | EXP 소량 | 모험 없이도 성장 가능 (느리지만 꾸준히) |
| 독서 완료 | 랜덤 스탯 +1 | 아들 기본 능력치 영구 상승 |

### 메타 → 코어 (강화된 환경이 전투력으로 변환)

| 메타 요소 | 코어에 미치는 영향 | 구체적 수치 |
|----------|------------------|-----------|
| 장비 제작 | 장비 기본 스탯 → 전투력 상승 | 슬롯별 STR/DEF/AGI/INT/HP 가산 |
| 장비 강화 (+1~+5) | 장비 스탯 증폭 → 전투력 추가 상승 | +10%~+80% 기본 스탯 가산 |
| 장비 등급 (가챠) | 등급 배율로 장비 효율 극대화 | x1.0~x2.2 배율 |
| 음식 제작 | 배고픔 회복 → 출발 조건 충족 단축 | 빵(+20)~영웅 도시락(+70) |
| 음식 임시 버프 | 1모험 동안 스탯 보너스 | 고기 스튜: STR +2, 도시락: 전 스탯 +1 |
| 포션 (체력) | HP 회복 → 생존 시간 연장 | HP +30 즉시 |
| 포션 (영약) | 1모험 스탯 버프 → 전투력 상승 | 각 스탯 +3~5 |
| 레시피 해금 | 상위 장비/음식/포션 접근 | 더 높은 스탯/효과의 아이템 제작 가능 |
| 농장 수확 | 요리/포션 재료 자급 → 지속 가능한 준비 | 모험 의존도 감소, 안정적 공급 |

### 순환 루프 요약

```
부모의 크래프팅 (메타)
   │
   ├─ 더 좋은 장비 ─┐
   ├─ 더 좋은 음식 ─┤
   ├─ 더 좋은 포션 ─┤
   │                ▼
   │      아들 전투력 상승 (코어)
   │                │
   │                ▼
   │      더 어려운 모험 성공 (코어)
   │                │
   │                ▼
   │      더 좋은 보상 획득 (코어 → 메타)
   │                │
   │                ▼
   └────── 더 좋은 재료 확보 (메타)
                    │
                    └─→ 다시 크래프팅으로 (반복)
```

**핵심 설계 원칙**: 이 순환이 매 사이클마다 **약간씩 상승**하도록 보상과 소비를 설계한다. 한 사이클의 보상이 다음 사이클에서 "한 단계 더 좋은" 장비/음식/포션을 만들기에 **거의 충분하지만 약간 부족**하도록 밸런싱하여, 플레이어가 "한 사이클만 더"를 반복하게 만든다.

### 상태 관리 가이드

메타 시스템에서 관리해야 할 전역 상태:

```typescript
// 인벤토리 상태
interface Inventory {
  gold: number;
  materials: Record<MaterialType, number>;  // 철광석, 가죽, 나무 등
  herbs: Record<HerbType, number>;          // 붉은/푸른/노란 약초
  monsterParts: Record<MonsterPartType, number>; // 이빨, 껍질
  seeds: Record<SeedType, number>;          // 각종 씨앗
  farmProduce: Record<ProduceType, number>; // 밀, 감자, 당근, 사과
  adventureLoot: {
    meat: number;                            // 고기 (모험 드롭 전용, 농장 불가)
  };
  enhancementStones: number;                // 강화석
  specialOre: number;                       // 특수 광석 (가챠용)
  mithril: number;                          // 미스릴
  gems: number;                             // 보석
  books: Book[];                            // 책 (독서용, 보스 드롭)
}

// 책 아이템
interface Book {
  name: string;
  statEffect: { stat: 'STR' | 'DEF' | 'AGI' | 'INT'; value: number }; // 독서 시 스탯 효과
}

// 해금 상태
interface UnlockState {
  recipes: Record<RecipeId, boolean>;       // 레시피 해금 여부
  systems: {
    alchemy: boolean;     // 연금술 (Lv.3)
    enhancement: boolean; // 강화 (Lv.5)
    gacha: boolean;       // 가챠 (Lv.8)
  };
  farmSlots: number;      // 농장 칸수 (4 → 6)
  potionSlots: number;    // 포션 선반 (3 → 5)
  milestones: Record<number, boolean>;  // 달성한 마일스톤 레벨
}

// 장비 상태
interface Equipment {
  id: string;
  name: string;
  slot: 'weapon' | 'armor' | 'accessory';
  rarity: 'common' | 'uncommon' | 'rare' | 'epic';
  baseStats: Stats;
  enhanceLevel: number;   // 0~5
  finalStats: Stats;      // 계산된 최종 스탯
}
```

### 밸런스 조정 포인트

프로토타입 플레이 테스트 후 아래 수치를 우선 조정한다:

| 조정 대상 | 조정 기준 | 방향 |
|----------|---------|------|
| 골드 드롭량 | 3사이클 차에 철 장비 1개 제작 가능해야 함 | 부족하면 ↑ |
| 강화석 드롭률 | 5사이클 차에 +2 강화 1회 가능해야 함 | 부족하면 ↑ |
| 농장 수확량 | 1사이클 동안 음식 2~3개 자급 가능해야 함 | 부족하면 수확량 ↑ |
| 배고픔 감소 속도 | 부모가 1개 이상 아이템 제작 후 출발하는 여유 | 빠르면 속도 ↓ |
| 모험 시간 | 부모가 다음 사이클 준비를 마칠 수 있는 시간 | 짧으면 시간 ↑ |
| 레시피 해금 레벨 | 3~5사이클 내에 2~3개 레시피 해금 체감 | 느리면 레벨 ↓ |
